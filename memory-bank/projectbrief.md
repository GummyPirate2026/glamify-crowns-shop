# Project Brief: Glamify Crowns Shop

## Project Overview
A self-hosted e-commerce platform inspired by the Glamifycrowns Etsy shop, designed to eliminate Etsy's listing and transaction fees while providing complete control over the shopping experience.

## Core Objective
Create a fully functional online store for selling handcrafted tiaras and crowns with:
- No platform fees
- Complete data ownership
- Full customization freedom
- Self-hosting capability

## Primary Goals

### 1. E-Commerce Functionality
- Product catalog with filtering and search
- Individual product detail pages
- Shopping cart with persistence
- Product management system

### 2. Admin Capabilities
- Product creation and editing
- Image upload from local computer
- Inventory management
- Basic sales statistics

### 3. User Experience
- Clean, modern design inspired by Glamifycrowns
- Mobile-responsive interface
- Fast product browsing
- Simple checkout process

## Scope

### In Scope (MVP)
- Product listings and detail pages
- Shopping cart
- Admin dashboard
- Image upload and storage
- Basic inventory tracking

### Future Enhancements (Optional)
- Payment processing integration
- User authentication
- Order management system
- Email notifications
- Customer reviews

## Success Criteria
- User can browse products
- User can add items to cart
- Admin can create/edit/delete products
- Admin can upload product images
- Site runs without external dependencies (except hosting)
- No ongoing platform fees

## Constraints
- Must be self-hostable
- Should work with local database
- Images stored with product data
- No reliance on external services for core functionality
